# PRD: 프로젝트 페이지 브레드크럼 명칭 통일 (2026-02-16)

## 목적
- 프로젝트 관련 페이지의 브레드크럼 명칭을 통일해 사용자 인지 부하를 줄인다.

## 적용 범위
- `frontend/src/pages/BudgetProjectOverview.jsx` (프로젝트 메인)
- `frontend/src/pages/BudgetProjectBudget.jsx` (예산 메인)

## 문제 정의
- 프로젝트 메인/예산 메인 브레드크럼 첫 경로가 `메인 / 글로벌 검색`으로 표시되어,
  다른 프로젝트 하위 페이지의 `프로젝트 관리` 명칭과 불일치한다.

## 요구사항
- REQ-001: 프로젝트 메인 브레드크럼 첫 항목을 `프로젝트 관리`로 표시한다.
- REQ-002: 예산 메인 브레드크럼 첫 항목을 `프로젝트 관리`로 표시한다.
- REQ-003: 기존 `메인 / 글로벌 검색` 표시는 제거한다.

## 수용 기준
- AC-001: 프로젝트 메인 브레드크럼이 `프로젝트 관리 > {프로젝트명}` 형태로 보인다.
- AC-002: 예산 메인 브레드크럼이 `프로젝트 관리 > {프로젝트명} > 예산 메인` 형태로 보인다.
