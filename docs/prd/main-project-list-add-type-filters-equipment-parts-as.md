# 메인 페이지 프로젝트 타입 필터(설비/파츠/AS) 추가 PRD

## 배경
- 메인 페이지 프로젝트 필터에 프로젝트 타입 기준 필터가 없어 목록 모니터링 범위를 빠르게 좁히기 어렵다.

## 목표
- 메인 페이지 프로젝트 필터에 `설비`, `파츠`, `AS` 항목을 추가하고 실제 목록 필터링에 반영한다.

## 요구사항
1. 필터 UI 추가
- 대상: `frontend/src/pages/SearchResults.jsx`
- 프로젝트 필터 영역에 타입 필터 버튼 그룹을 추가한다.
- 항목: `설비`, `파츠`, `AS` (복수 선택 가능)

2. 필터 동작
- 선택된 타입 값만 프로젝트 리스트에 노출한다.
- 타입 필터 미선택 시 전체 타입을 노출한다.
- `project_type`, `project_type_label` 모두 대응 가능하도록 정규화 로직을 둔다.

## 완료 기준
- 메인 페이지에서 설비/파츠/AS 필터 버튼이 노출된다.
- 타입 필터 선택 시 리스트가 정상적으로 필터링된다.
- Docker 환경 `verify:fast`가 통과한다.
