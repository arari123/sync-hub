# 예산 메인 트리/필터 개선 PRD

## 배경
- 예산 메인의 상세 트리 정렬/가시성이 낮고, 검색/필터 동작 요구사항이 현재 UX와 맞지 않는다.

## 목표
1. 트리 시인성 개선(들여쓰기 제거, 열 정렬 유지, 레벨별 색/타이포 구분)
2. 필터/검색 기능 확장(전체 선택/해제, 통합 검색, 검색어 하이라이트, 자동 펼침)
3. 상단 상세 패널 조작성 강화(모두 접기/펼치기)
4. 합계 금액 고정 기능 제공

## 요구사항
1. 재료비 상세에서 유닛/예산/집행/상태 헤더 제거
2. 트리 구조는 들여쓰기 없이 레벨별 시각 구분으로 표현
3. `예산 상세 브레이크다운` -> `예산 상세`로 변경 및 `모두 접기/펼치기` 토글 제공
4. 사이드바에 `합계 금액 고정` 옵션 추가
- 고정 시 필터링과 무관하게 합계는 전체 선택 기준 금액 유지
5. 필터에 `전체 선택`, `전체 해제` 버튼 제공
- 전역 버튼 + 프로젝트 단계/비용 유형/설비 항목별 버튼
6. 사이드바 검색창은 예산 내역 전체 검색
- 선택된 필터 범위 내에서만 검색
- 검색어 하이라이트
- 검색으로 매칭된 접힌 트리 내용은 자동 펼침

## 완료 기준
- 상기 요구사항이 예산 메인(`/project-management/projects/:projectId/budget`)에 반영된다.
- Docker 검증 통과:
  - `docker-compose exec -T frontend sh -lc 'cd /app && npm run build'`
  - `docker-compose exec -T web bash -lc 'cd /app && bash scripts/verify_fast.sh'`
