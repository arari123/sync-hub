# 프로젝트 메인 상단 단계 세그먼트/메인 배지 제거 PRD

## 배경
- 프로젝트 메인 본문 상단에 우측 단계 세그먼트(검토/설계/...)와 제목 옆 `프로젝트 메인` 배지가 중복 노출되고 있다.

## 목표
- 프로젝트 정보 헤더의 우측 단계 세그먼트를 제거한다.
- 프로젝트 상태 배지 오른쪽 `프로젝트 메인` 문구를 제거한다.

## 요구사항
1. 우측 단계 세그먼트 제거
- 대상: `frontend/src/pages/BudgetProjectOverview.jsx`
- 프로젝트 정보 카드 상단 우측의 단계 세그먼트 렌더링 블록을 삭제한다.

2. 제목 옆 메인 배지 제거
- 대상: `frontend/src/pages/BudgetProjectOverview.jsx`
- 프로젝트 상태 배지 옆 `프로젝트 메인` 배지를 삭제한다.

## 완료 기준
- 우측 단계 세그먼트가 보이지 않는다.
- 프로젝트 상태 오른쪽 `프로젝트 메인` 문구가 보이지 않는다.
- Docker 환경 `verify:fast`가 통과한다.
