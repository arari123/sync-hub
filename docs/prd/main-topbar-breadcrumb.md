# 메인 상단바 브레드크럼 표시 PRD

## 배경
- 메인 페이지 상단바에 현재 로고, 검색 입력, 액션 아이콘은 있으나 현재 위치를 나타내는 경로 정보가 없다.

## 목표
- 상단바에 브레드크럼 경로를 표시한다.
- 브레드크럼의 시작 위치는 검색어 입력 시작 위치와 동일하게 맞춘다.

## 요구사항
1. 상단바 브레드크럼 추가
- 대상: `frontend/src/pages/SearchResults.jsx`
- 검색 입력 영역 상단에 브레드크럼(`메인 / 글로벌 검색`)을 표시한다.

2. 정렬 기준
- 브레드크럼의 좌측 시작점을 검색 입력창의 좌측 시작점과 동일하게 유지한다.

3. 레이아웃 보정
- 브레드크럼 추가로 인해 상단 높이 및 본문 최소 높이 계산이 필요한 경우 함께 조정한다.

## 완료 기준
- `http://localhost:8000/`에서 상단바에 브레드크럼이 표시된다.
- 브레드크럼 시작선과 검색 입력 시작선이 맞는다.
- Docker 환경 `verify:fast`가 통과한다.
