# 프로젝트 메인 브레드크럼 패널 메뉴 확장 PRD

## 배경
- 프로젝트 메인 페이지의 브레드크럼 패널에는 현재 경로만 표시되고, 프로젝트 하위 이동 메뉴가 없다.
- 사용자는 브레드크럼 패널에 프로젝트 섹션 메뉴를 추가하고, 예산 관리는 하위 메뉴(탑다운)로 확장되길 원한다.

## 목표
- 프로젝트 메인 페이지 브레드크럼 패널에 섹션 메뉴를 추가한다.
- 예산 관리는 드롭다운(탑다운) 하위 메뉴로 제공한다.
- 메뉴 정렬 기준을 상단바 우측 유저 아이콘 끝선과 일치시킨다.

## 요구사항
1. 메뉴 구성
- 프로젝트 메인
- 예산 관리 (드롭다운: 재료비 관리, 인건비 관리, 경비 관리)
- 이슈 관리
- 일정 관리
- 사양 관리
- 데이터 관리
- 프로젝트 설정

2. 정렬
- 브레드크럼 패널 내 메뉴의 마지막 버튼(`프로젝트 설정`) 우측 끝은 상단바 유저 아이콘 우측 끝과 일치해야 한다.

3. 노출 조건
- 해당 메뉴는 프로젝트 메인 페이지(`/project-management/projects/:projectId`)에서만 노출된다.

## 완료 기준
- 프로젝트 메인 페이지에서 브레드크럼 패널 메뉴가 노출된다.
- 예산 관리 드롭다운이 동작한다.
- 정렬 기준이 상단바 우측 아이콘 끝선과 시각적으로 일치한다.
- Docker 환경 `verify:fast`가 통과한다.
