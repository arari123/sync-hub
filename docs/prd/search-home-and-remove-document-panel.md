# PRD: 검색 페이지 홈 전환 및 문서 패널 제거

## 목적
업데이트된 검색 대시보드 페이지를 서비스의 기본 홈(`/`)으로 전환하고, 해당 페이지 하단의 문서 검색 결과 패널을 제거한다.

## 요구사항

### SHDP-REQ-001: 홈 전환
- [AC-1] 루트 경로(`/`)에서 검색 대시보드 페이지가 직접 렌더링되어야 한다.
- [AC-2] 기존 `/search` 접근은 호환성을 위해 `/`로 리다이렉트되어야 한다.
- [AC-3] 검색 대시보드 전용 상단/레이아웃은 `/`에서도 동일하게 적용되어야 한다.

### SHDP-REQ-002: 문서 패널 제거
- [AC-1] 검색 페이지의 문서 검색 결과 섹션(UI 패널)을 제거한다.
- [AC-2] 프로젝트 목록/필터/검색 동작은 유지한다.

## 범위
- `frontend/src/App.jsx` 라우팅 수정
- `frontend/src/components/Layout.jsx` 경로 조건 수정
- `frontend/src/pages/SearchResults.jsx` 문서 패널 제거

## 비범위
- 백엔드 문서 검색 API 변경
- 프로젝트 상세/생성 페이지 변경
