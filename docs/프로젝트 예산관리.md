# 프로젝트 예산관리 기능 기획 요약

## 1. 목적
- PM이 설비 제작/설치 프로젝트의 예산을 단계별로 관리하고, 확정 이후 변경 이력까지 추적할 수 있는 웹 기능을 구축한다.

## 2. 핵심 방향
- 버전 컨트롤 타워: 단계별(`검토`, `제작`, `설치`, `워런티`, `종료`) 확정 이력 관리
- 비용 구조 표준화: 재료비/인건비/경비 + 제작/설치 축 분리
- 엑셀 왕복 처리: 템플릿 다운로드, 수정 업로드, 수식 유지
- 결재 연계: 웹 내 결재 미구현, PDF 내보내기로 그룹웨어 결재 대응

## 3. 문서 구성
- 상세 PRD: `docs/prd/budget-management.md`
- 실행 계획: `.agent/execplans/2026-02-07-budget-management-entry.md`

## 4. 현재 구현 상태(Phase 1)
- 검색 페이지 상단에서 `프로젝트 예산관리` 버튼으로 예산관리 페이지(`/project-management`) 이동 가능
- 프로젝트 생성/조회 API 구현
- 버전 생성(`draft`), 확정(`confirmed`), 리비전(`revision`) 생성 API 구현
- 버전별 설비 예산(재료/인건비/경비, 제작/설치) 입력/저장 API 구현
- 예산관리 페이지에서:
  - 프로젝트 선택
  - 버전 생성/확정/리비전 생성
  - 설비 예산 행 편집 및 저장
  - 합계(재료비/인건비/경비/제작비/설치비/총액) 표시

## 5. 다음 구현 우선순위
1. 예산 요약 대시보드 고도화(설비별 비중 차트, 단계별 비교)
2. 버전 비교 화면(추가/변경/삭제 하이라이트) 구현
3. 엑셀 템플릿 내보내기/재업로드 파이프라인 구현
4. PDF 내보내기(외부 결재 제출용) 구현
