# 메인 프로젝트 리스트 클론 레이아웃 및 프로젝트 내 검색 PRD

## 배경
- 메인 페이지의 프로젝트 필터/리스트 패널을 `docs/ex_project_list.html` 기준 시안과 동일한 구조로 개편할 필요가 있다.
- 필터 패널의 검색 입력은 전역 검색이 아니라 프로젝트 데이터 범위 내 단어 매칭 검색으로 동작해야 한다.

## 목표
- 메인 페이지 프로젝트 필터/리스트 패널을 시안 레이아웃으로 클론 구현한다.
- 시안의 영문 항목명은 한국어로 번역하여 표기한다.
- 필터 패널 검색 입력이 프로젝트 데이터 필드 대상 단어 매칭으로 동작하게 한다.

## 요구사항
1. 패널 레이아웃 클론
- 대상: `frontend/src/pages/SearchResults.jsx`
- 기존 테이블 기반 프로젝트 리스트를 시안 구조(상단 필터 바 + 카드형 프로젝트 리스트)로 교체한다.
- 항목명 번역: My Projects/All Projects/All Stages/State/Type/Recent Agendas/Search projects 등 영문 문구를 한국어로 표기한다.

2. 프로젝트 내 검색 기능
- 필터 패널의 검색 입력을 추가한다.
- 검색은 프로젝트 리스트 데이터 범위에서만 동작해야 하며, 입력 단어가 프로젝트 필드(이름/코드/설명/고객사/담당자/위치/설비명/상태/종류)에 매칭될 때만 노출한다.
- 기존 전역 검색(상단 글로벌 검색)은 기존대로 유지한다.

## 완료 기준
- 필터/리스트 패널이 `docs/ex_project_list.html` 구조와 유사한 카드형 레이아웃으로 동작한다.
- 패널 내 영문 항목명이 한국어로 표시된다.
- 프로젝트 내 단어 매칭 검색이 정상 동작한다.
- Docker 환경 `verify:fast`가 통과한다.
