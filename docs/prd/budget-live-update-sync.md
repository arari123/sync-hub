# PRD: 예산 입력/조회 실시간 동기화 (2026-02-13)

## 배경
- 예산 입력 모드에서 항목 추가/수정/삭제 후 저장해도 예산 메인 상위 집계 패널과 조회 페이지에 즉시 반영되지 않고 새로고침이 필요하다.
- 입력 화면과 조회 화면 간 동기화 타이밍이 분리되어 사용자 혼란이 발생한다.

## 목표
- 예산 입력 모드에서 변경한 값이 예산 메인 집계 패널에 실시간 반영된다.
- 저장 이후 프로젝트 메인/조회 페이지가 자동으로 최신 데이터를 다시 로드한다.

## 범위
- 포함:
  - `BudgetProjectEditor` -> `BudgetProjectBudget` 실시간 상세 데이터 동기화
  - 저장/확정/리비전 이벤트 발생 시 조회 페이지 자동 재조회
- 제외:
  - 백엔드 집계 로직 변경
  - 다중 브라우저 탭 간 실시간 동기화(WebSocket)

## 요구사항
1. 입력 모드에서 행 추가/수정/삭제 시 상단 집계 카드 값이 즉시 바뀌어야 한다.
2. 상세 저장/버전 확정/확정 취소/리비전 생성 후 조회 페이지가 새로고침 없이 최신 집계로 갱신되어야 한다.
3. 프로젝트가 다른 페이지에는 불필요한 재조회가 발생하지 않아야 한다.

## 완료 기준
- 프론트 빌드 성공
- `verify:fast` 통과
- 예산 입력 -> 저장 후 예산 메인/프로젝트 메인에서 집계 값 자동 갱신 확인
