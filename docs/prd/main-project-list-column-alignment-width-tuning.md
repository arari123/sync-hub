# 메인 페이지 프로젝트 리스트 정렬/열 너비 튜닝 PRD

## 배경
- 메인 페이지 프로젝트 리스트에서 일부 열의 정렬이 일관되지 않고, 핵심 열(프로젝트/마지막 안건)에 더 많은 공간이 필요하다.

## 목표
- 지정된 열을 중앙 정렬로 통일한다.
- 고객사/위치, 담당자, 프로젝트 상태, 프로젝트 종류 열 폭을 줄이고 프로젝트/마지막 안건 열 폭을 확장한다.

## 요구사항
1. 중앙 정렬
- 대상: `frontend/src/pages/SearchResults.jsx`
- `고객사/위치`, `담당자`, `프로젝트 상태`, `프로젝트 종류`, `ACTION` 열의 헤더/셀을 중앙 정렬한다.

2. 열 너비 조정
- `고객사/위치`, `담당자`, `프로젝트 상태`, `프로젝트 종류` 열 폭 최소화
- 줄어든 폭을 `프로젝트`, `마지막 안건` 열에 반영하여 확장

## 완료 기준
- 지정 열 중앙 정렬이 반영된다.
- 프로젝트/마지막 안건 열이 기존보다 넓게 표시된다.
- Docker 환경 `verify:fast`가 통과한다.
