# 프로젝트 메인 예산 하위 메뉴 우측 여백 축소 PRD

## 배경
- 예산 하위 메뉴 드롭다운에 고정 최소폭이 적용되어 경비 관리 우측에 불필요한 빈 공간이 발생한다.

## 목표
- 하위 메뉴 패널 폭을 콘텐츠 기준으로 조정해 우측 빈 공간을 제거한다.
- 3개 하위 메뉴 1행 노출은 유지한다.

## 요구사항
1. 폭 조정
- 대상: `frontend/src/pages/BudgetProjectOverview.jsx`
- 예산 하위 메뉴 컨테이너의 고정 최소폭 스타일을 제거한다.

2. 레이아웃 유지
- 하위 메뉴 항목은 줄바꿈 없이 1행으로 유지한다.

## 완료 기준
- 경비 관리 우측 과도한 공백이 제거된다.
- Docker 환경 `verify:fast`가 통과한다.
