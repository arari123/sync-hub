# 프로젝트 메인 브레드크럼 메뉴 hover 개선 PRD

## 배경
- 프로젝트 메인 브레드크럼 패널 메뉴가 검색 페이지 필터 세그먼트 디자인과 다르다.
- 예산 관리 하위 메뉴는 클릭 기반 열림이며, 사용자 요구는 hover 기반 노출과 1초 유지다.

## 목표
- 프로젝트 메인 브레드크럼 패널 메뉴를 검색 페이지 필터와 동일한 세그먼트 스타일로 통일한다.
- 예산 관리는 hover 시 하위 메뉴가 노출되고, 버튼 클릭 시 예산 관리 페이지로 이동되도록 한다.

## 요구사항
1. 디자인 정렬
- 대상: `frontend/src/pages/BudgetProjectOverview.jsx`
- 브레드크럼 패널 메뉴(프로젝트 메인~프로젝트 설정)를 검색 페이지 필터 세그먼트 디자인으로 변경.

2. 예산 관리 동작
- 예산 관리 버튼은 `hover` 시 하위 메뉴 노출.
- 예산 관리 버튼 클릭 시 `/budget` 페이지로 이동.
- 하위 메뉴는 1열이 아닌 1행 레이아웃으로 표시.
- 마우스 이탈 후 1초 동안 하위 메뉴 유지 후 닫힘.

## 완료 기준
- 메뉴 디자인이 검색 필터와 동일한 스타일 계열로 적용된다.
- 예산 관리 hover/클릭 동작과 1초 유지 동작이 모두 충족된다.
- Docker 환경 `verify:fast`가 통과한다.
