# 메인 필터 단계/유형 활성 가시성 개선 PRD

## 배경
- 메인 페이지에서 `내 프로젝트/전체 프로젝트`를 제외한 단계/유형 필터는 활성/비활성 시각 대비가 약해 상태 인지가 어렵다.

## 목표
- 단계/유형 필터의 활성/비활성 스타일 대비를 강화해 선택 상태를 즉시 구분 가능하게 한다.

## 요구사항
1. 대상 범위
- `frontend/src/pages/SearchResults.jsx`
- 단계 필터(전체 단계 + 단계 버튼)
- 유형 필터(전체 유형 + 설비/파츠/AS 버튼)

2. 스타일 정책
- 활성: 강한 채움 색(Primary) + 고대비 텍스트
- 비활성: 중립 배경/테두리 + hover 상태만 강조
- `내 프로젝트/전체 프로젝트` 토글 스타일은 변경하지 않는다.

## 완료 기준
- 단계/유형 필터에서 활성/비활성 상태가 명확히 구분된다.
- Docker 환경 `verify:fast`가 통과한다.
