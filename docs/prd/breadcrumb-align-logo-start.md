# 브레드크럼 시작 위치 정렬 PRD

## 배경
- 현재 상단바 하단 브레드크럼 패널은 좌측 스페이서(`w-44`)를 사용해 검색 입력 시작점에 정렬되어 있다.
- 요청사항은 브레드크럼 시작 위치를 로고 시작 위치(컨테이너 좌측 패딩 기준)로 변경하는 것이다.

## 목표
- 브레드크럼 시작점을 로고 시작 위치와 일치시킨다.
- 메인 검색 페이지와 프로젝트 메인 페이지 모두 동일한 정렬 규칙을 적용한다.

## 요구사항
1. 검색 페이지 브레드크럼 정렬 변경
- 대상: `frontend/src/pages/SearchResults.jsx`
- 브레드크럼 패널의 좌/우 스페이서 제거.

2. 프로젝트 메인 페이지 브레드크럼 정렬 변경
- 대상: `frontend/src/pages/BudgetProjectOverview.jsx`
- 브레드크럼 패널의 좌/우 스페이서 제거.

## 완료 기준
- 두 페이지 브레드크럼 시작 위치가 로고 시작 위치와 동일하다.
- Docker 환경에서 `verify:fast`가 통과한다.
