# PRD: 안건 리스트 진행상태(진행중/종료) 표시 추가 (2026-02-16)

## 목적
- 안건 리스트에서 각 항목의 처리 상태를 빠르게 식별할 수 있도록 `진행중/종료` 상태를 노출한다.

## 적용 범위
- 공용 안건 리스트 컴포넌트: `frontend/src/components/agenda/AgendaSplitView.jsx`
- 메인 `전체안건` 및 프로젝트 `안건 관리` 리스트 공통 적용

## 요구사항
- REQ-001: 안건 리스트 각 항목에 `진행중/종료` 상태를 표시한다.
- REQ-002: 상태는 기존 타입 배지(최초등록/답변)와 함께 상단 메타 영역에서 보여준다.
- REQ-003: 상태값 `completed`는 `종료`, 그 외는 `진행중`으로 표기한다.

## 수용 기준
- AC-001: 메인 안건 리스트에서 상태 배지가 보인다.
- AC-002: 프로젝트 안건 리스트에서 상태 배지가 보인다.
- AC-003: `completed` 항목은 `종료`, 진행 항목은 `진행중`으로 확인된다.
