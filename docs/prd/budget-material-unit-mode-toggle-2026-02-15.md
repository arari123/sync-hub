# PRD: 예산 메인 재료비 탭 유닛 표시 모드(예산/집행) 전환 (2026-02-15)

## 목적
- 예산 메인(`프로젝트 > 예산`) 재료비 탭 조회 화면에서 프로젝트 단계에 맞는 유닛(예산/집행)을 기본으로 노출한다.
- 검토/진행 단계 모두에서 `예산/집행` 유닛을 전환하여 확인할 수 있어야 한다.

## 범위
- 프론트엔드: `frontend/src/pages/BudgetProjectBudget.jsx`
- 예산 메인 탭 중 `재료비`(조회 화면, 입력 모드 제외)

## 요구사항
- REQ-001: 프로젝트 단계가 `검토(review)`이면 기본 표시 모드는 `예산 유닛`이다.
- REQ-002: 프로젝트 단계가 `검토(review)`가 아니면 기본 표시 모드는 `집행 유닛`이다.
- REQ-003: 재료비 탭 조회 화면에 `예산 유닛/집행 유닛` 전환 버튼을 제공한다.
  - 어떤 단계(검토/진행)에서도 전환 가능해야 한다.
- REQ-004: `예산 유닛` 모드에서는 예산 입력 기반 유닛 목록만 표시한다.
- REQ-005: `집행 유닛` 모드에서는 집행 입력 기반 유닛 목록만 표시한다.

## 수용 기준
- AC-001: 검토 단계 프로젝트에서 재료비 탭을 열면 예산 유닛 화면이 기본으로 보인다.
- AC-002: 검토가 아닌 단계 프로젝트에서 재료비 탭을 열면 집행 유닛 화면이 기본으로 보인다.
- AC-003: 전환 버튼으로 예산/집행 유닛을 즉시 교체하여 확인할 수 있다.
