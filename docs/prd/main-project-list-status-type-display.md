# 메인 페이지 프로젝트 리스트 상태/종류 표시 PRD

## 배경
- 메인(글로벌 검색) 페이지의 프로젝트 리스트에서 프로젝트 상태와 프로젝트 종류를 한눈에 확인하기 어렵다.

## 목표
- 프로젝트 리스트에 `프로젝트 상태`, `프로젝트 종류` 정보를 명시적으로 표시한다.

## 요구사항
1. 표시 항목 추가
- 대상: `frontend/src/pages/SearchResults.jsx`
- 프로젝트 테이블에 `프로젝트 상태`, `프로젝트 종류` 열을 추가한다.

2. 라벨 보정
- API에서 라벨 필드가 없더라도 코드값으로 한국어 라벨을 계산해 표시한다.

## 완료 기준
- 메인 페이지 프로젝트 리스트에서 상태/종류가 각 행에 표시된다.
- Docker 환경 `verify:fast`가 통과한다.
