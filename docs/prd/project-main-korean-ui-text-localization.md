# 프로젝트 메인 영문 UI 한글화 PRD

## 배경
- 프로젝트 메인 화면에 영문 UI 문구가 다수 노출되고 있다.
- 운영 규칙상 사용자 노출 문구는 한국어 기준으로 일관되게 제공되어야 한다.

## 목표
- 프로젝트 메인 화면에서 사용자에게 보이는 영문 텍스트를 한글로 변경한다.

## 요구사항
1. 대상 화면
- `frontend/src/pages/BudgetProjectOverview.jsx`

2. 한글화 범위
- 상단/패널 제목/버튼/상태 라벨/통계 카드/단계 라벨 등 화면 노출 영문 문구 전체를 한글로 전환한다.
- 약어는 한국어 표기 우선으로 적용한다(예: NO-CODE -> 코드 없음).

## 완료 기준
- 프로젝트 메인 화면의 영문 노출 텍스트가 한글로 변경된다.
- Docker 환경 `verify:fast`가 통과한다.
