# PRD: 예산 통합 요약 탭 인건비/경비 단계별 상세 펼침 지원 (2026-02-15)

## 목적
- 예산 메인(`프로젝트 > 예산`)의 `통합 요약` 탭에서 재료비와 동일한 UX로 인건비/경비도 `제작/설치` 단계 클릭 시 상세 항목을 펼쳐 볼 수 있게 한다.

## 배경/문제
- 현재 통합 요약의 “통합 원가 상세” 테이블은 재료비만 단계 클릭 시 상세(유닛/파츠)를 펼칠 수 있고, 인건비/경비는 합계만 보여서 단계별 상세 확인이 불편하다.

## 범위
- 프론트엔드: `frontend/src/pages/BudgetProjectBudget.jsx`
- 통합 요약 탭의 “통합 원가 상세” 테이블(비용군/단계 트리)

## 요구사항
- REQ-001: 통합 요약 탭에서 `인건비`의 `제작/설치` 단계 행을 클릭하면 해당 단계의 상세 항목이 하위 행으로 펼쳐져야 한다.
- REQ-002: 통합 요약 탭에서 `경비`의 `제작/설치` 단계 행을 클릭하면 해당 단계의 상세 항목이 하위 행으로 펼쳐져야 한다.
- REQ-003: 펼침 UI는 재료비와 동일한 형식(단계 행의 화살표/카운트 배지 + 하위 행 리스트)으로 제공한다.

## 수용 기준
- AC-001: 통합 요약 탭의 인건비 `제작/설치` 단계 행에 화살표/항목 수 배지가 표시된다.
- AC-002: 인건비 `제작/설치` 단계 행을 클릭하면 상세 항목이 하위 행으로 나타나고, 다시 클릭하면 접힌다.
- AC-003: 통합 요약 탭의 경비 `제작/설치` 단계 행에 화살표/항목 수 배지가 표시된다.
- AC-004: 경비 `제작/설치` 단계 행을 클릭하면 상세 항목이 하위 행으로 나타나고, 다시 클릭하면 접힌다.
