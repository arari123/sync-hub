# PRD: 프로젝트 데이터 관리 업로드 후 화면 깜빡임(반복 새로고침 체감) 수정

## 배경
프로젝트 데이터 관리 페이지에서 파일 업로드 후 문서 상태가 `pending/processing`인 동안 자동 폴링이 동작한다.
현재 구현은 폴링 요청마다 파일 목록 로딩 상태(`isFileLoading`)를 켜서 목록 영역이 반복적으로 로딩 UI로 전환되며 화면이 깜빡이는 문제가 있다.

## 요구사항
1. 업로드 후 상태 폴링은 유지한다.
2. 폴링 요청은 UI 로딩 상태를 토글하지 않는 `silent` 갱신으로 수행한다.
3. 수동 조회/검색/페이지 이동/폴더 이동 등 사용자가 명시적으로 발생시킨 조회만 로딩 UI를 표시한다.

## 완료 기준
- 업로드 직후 상태 갱신은 계속되지만, 파일 목록 영역이 주기적으로 로딩 상태로 바뀌지 않는다.
- 기존 파일 목록/검색/페이지네이션 동작은 유지된다.
