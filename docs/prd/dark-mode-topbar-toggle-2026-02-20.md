# PRD: 다크모드 전환 버튼 추가 (2026-02-20)

## 배경
- 현재 UI에 다크모드 CSS 변수/클래스가 일부 존재하지만 사용자 전환 경로가 없다.
- 사용자 요청: 상단바에서 유저 아이콘이 보이는 영역에서 다크/라이트를 전환하고 싶다.

## 요구사항
- REQ-001: 사용자 메뉴가 노출되는 상단바에 다크모드 전환 버튼을 제공한다.
- REQ-002: 버튼 클릭 시 즉시 라이트/다크 테마가 전환되어야 한다.
- REQ-003: 테마 선택은 브라우저 재접속 후에도 유지되어야 한다.
- REQ-004: 초기 로딩 시 저장된 테마가 즉시 적용되어 화면 깜빡임을 최소화해야 한다.
- REQ-005: 다크모드에서 전역 배경(바디 배경/그리드 오버레이)이 다크 톤으로 자연스럽게 표시되어야 한다.

## 범위
- 포함:
  - 상단바 토글 버튼 UI 추가
  - 테마 상태 관리(localStorage + html.dark)
  - 초기 테마 적용 로직
  - 전역 배경 다크 스타일 보강
- 제외:
  - 사용자별 서버 저장
  - 테마 색상 팔레트 전체 재설계

## 완료 기준
- 상단바 버튼으로 다크/라이트 전환 가능
- 새로고침 후 선택한 테마 유지
- Docker 기준 `verify:fast` 통과
