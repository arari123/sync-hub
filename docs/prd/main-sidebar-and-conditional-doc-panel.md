# PRD: 메인 사이드바 라벨 및 조건부 문서 패널 노출

## 목적
홈 검색 대시보드 페이지에서 사이드바 현재 위치 라벨을 `메인`으로 맞추고, 문서 검색 결과 패널은 검색 수행 시에만 노출한다.

## 요구사항

### MSDP-REQ-001: 사이드바 현재 위치
- [AC-1] 홈 대시보드 페이지의 활성 메뉴 라벨은 `메인`이어야 한다.

### MSDP-REQ-002: 조건부 문서 패널
- [AC-1] 검색어가 있을 때만 `문서 검색 결과` 패널이 노출되어야 한다.
- [AC-2] 검색어가 없을 때는 문서 패널이 보이지 않아야 한다.
- [AC-3] 문서 검색 결과 패널은 기존 문서 검색 API 결과를 표시해야 한다.
