# 프로젝트 메인 정보 필드 굵기/코드 줄바꿈 PRD

## 배경
- 프로젝트 메인 정보 필드에서 값 텍스트 강조가 부족하고, 프로젝트 코드 표시는 라벨 하단 줄바꿈 표시를 명확히 할 필요가 있다.

## 목표
- 고객사/위치/담당자/프로젝트 코드 값 텍스트를 굵게 표시한다.
- 프로젝트 코드가 라벨 아래 다음 줄에 확실히 표시되도록 레이아웃을 보정한다.

## 요구사항
1. 값 텍스트 굵기 강화
- 대상: `frontend/src/pages/BudgetProjectOverview.jsx`
- 정보 필드 값(고객사, 위치, 담당자, 프로젝트 코드)을 굵은 텍스트로 렌더링한다.

2. 프로젝트 코드 줄바꿈 보장
- 라벨과 값의 줄 분리를 확실히 보장한다.
- 프로젝트 코드 값은 다음 행에서 보이도록 스타일을 정리한다.

## 완료 기준
- 고객사/위치/담당자/프로젝트 코드 값이 굵게 보인다.
- 프로젝트 코드가 라벨 다음 행에 표시된다.
- Docker 환경 `verify:fast`가 통과한다.
