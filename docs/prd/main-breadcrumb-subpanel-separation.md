# 메인 브레드크럼 서브패널 분리 PRD

## 배경
- 현재 브레드크럼이 상단바 내부에 포함되어 있어, 상단바 정보(로고/검색/아이콘)와 경로 정보의 시각적 구분이 약하다.

## 목표
- 브레드크럼을 상단바에서 분리한다.
- 상단바 하단의 종속 패널로 브레드크럼을 이동한다.
- 브레드크럼 패널 배경색을 상단바와 구분되도록 적용한다.

## 요구사항
1. 브레드크럼 분리
- 대상: `frontend/src/pages/SearchResults.jsx`
- 상단바 내부 브레드크럼을 제거하고, 상단바 바로 아래 별도 패널에 렌더링한다.

2. 시각적 구분
- 상단바와 브레드크럼 패널의 배경색을 다르게 적용한다.
- 두 영역은 경계선으로도 구분한다.

3. 기존 동작 유지
- 브레드크럼 클릭 이동 기능은 유지한다.
- 검색 입력과 우측 액션 아이콘 동작은 기존과 동일해야 한다.

## 완료 기준
- 브레드크럼이 상단바 외부의 하단 패널로 이동되어 보인다.
- 패널 배경색이 상단바와 명확히 구분된다.
- Docker 환경에서 `verify:fast`가 통과한다.
