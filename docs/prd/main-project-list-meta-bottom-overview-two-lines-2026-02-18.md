# PRD: 메인 프로젝트 리스트 메타 하단 배치 및 개요 2줄 표시 (2026-02-18)

## 배경
- 메인 페이지 프로젝트 리스트 카드에서 고객사/설치장소/담당자 정보가 개요 바로 아래에 붙어 보여 가독성이 낮다.
- 카드에서 프로젝트 개요가 충분히 보이지 않아 최소 2줄 표시가 필요하다.
- 리스트 패널 전체 크기는 유지되어야 한다.

## 목표
- 고객사/설치장소/담당자 메타 정보를 카드 하단으로 이동한다.
- 프로젝트 개요는 카드에서 2줄까지 안정적으로 보이도록 표시한다.
- 기존 리스트 패널 레이아웃과 크기 체감은 유지한다.

## 범위
- 프론트엔드 메인 페이지 프로젝트 리스트 카드 레이아웃 조정

## 비범위
- 프로젝트 상세 페이지 레이아웃 변경
- 검색 결과 페이지(문서/안건) 카드 UI 변경
- 리스트 패널 폭/높이 규격 자체 변경

## 요구사항
- REQ-001: 메인 프로젝트 카드에서 고객사/설치장소/담당자 메타 정보는 카드 하단 영역에 배치한다.
- REQ-002: 프로젝트 개요 텍스트는 최대 2줄까지 표시되고 넘치는 텍스트는 잘린다.
- REQ-003: 빠른 검증(`verify:fast`)을 Docker 환경에서 통과한다.

## 완료 기준
- 메인 페이지 프로젝트 리스트 카드에서 메타 정보가 하단으로 이동한 것이 확인된다.
- 프로젝트 개요가 2줄 표시 규칙으로 렌더링된다.
- Docker 환경 `verify:fast` 통과
