# 메인 검색 필터 단계 디자인 정렬 PRD

## 배경
- 프로젝트 메인 페이지의 단계 표시 UI와 메인/글로벌 검색 페이지 필터 버튼 UI가 서로 다른 디자인을 사용하고 있다.
- 사용자 요청은 프로젝트 메인 단계 UI 디자인을 검색 페이지 필터 버튼에 적용하는 것이다.

## 목표
- 검색 페이지 필터 버튼(내/전체 토글 + 단계 필터)을 프로젝트 메인 단계 표시 디자인과 동일한 톤으로 통일한다.

## 요구사항
1. 대상 페이지
- `frontend/src/pages/SearchResults.jsx`

2. 적용 범위
- `내 프로젝트`, `전체 프로젝트` 토글 버튼
- `전체 단계` 및 단계 선택 버튼 전체

3. 디자인 기준
- 프로젝트 메인 단계 UI의 세그먼트 스타일 적용
  - 그룹 컨테이너: `bg-secondary`, `p-1`, `rounded-lg`, `inline-flex/flex-wrap`
  - 활성 버튼: `bg-primary`, `text-primary-foreground`, `shadow-sm`
  - 비활성 버튼: muted 텍스트 + hover 상태

## 완료 기준
- 검색 페이지 필터 버튼이 프로젝트 메인 단계 표시와 동일한 스타일 계열로 보인다.
- 기존 필터 동작(토글/다중선택)은 유지된다.
- Docker 환경 `verify:fast`가 통과한다.
