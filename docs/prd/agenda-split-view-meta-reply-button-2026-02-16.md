# PRD: 안건 Split View 메인 프로젝트 메타/답변 작성 버튼 보강 (2026-02-16)

## 목적
- 메인 `전체안건` 리스트에서 안건의 소속 프로젝트를 더 빠르게 식별할 수 있게 한다.
- Split View 우측 상세에서 즉시 답변 작성으로 진입할 수 있게 한다.

## 적용 범위
- `frontend/src/components/agenda/AgendaSplitView.jsx`
- `frontend/src/pages/AgendaDetail.jsx`

## 요구사항
- REQ-001: 메인 페이지 안건 리스트(`mode=my`)에서만 제목 아래 `프로젝트 코드 · 프로젝트 이름`을 표시한다.
- REQ-002: 프로젝트 안건 리스트(`mode=project`)에서는 프로젝트 메타를 표시하지 않는다.
- REQ-003: 우측 안건 상세 헤더에 `답변 작성` 버튼을 추가한다.
- REQ-004: `답변 작성` 버튼 클릭 시 안건 상세 페이지 답변 작성 UI가 열린 상태로 이동한다.

## 수용 기준
- AC-001: 메인 리스트에만 프로젝트 코드/이름이 보인다.
- AC-002: 프로젝트 리스트에는 프로젝트 코드/이름이 보이지 않는다.
- AC-003: Split View 우측에 `답변 작성` 버튼이 보이고, 클릭 시 답변 폼이 열린 상세 페이지로 이동한다.
