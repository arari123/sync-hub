# 메인 필터 단계/유형 사이 구분선 추가 PRD

## 배경
- 메인 페이지 필터에서 `전체 프로젝트↔전체 단계` 사이에는 구분선이 있으나, `종료↔전체 유형` 사이에는 구분선이 없어 시각적 구획이 불명확하다.

## 목표
- `전체 단계`(종료 포함) 그룹과 `전체 유형` 그룹 사이에 동일한 세로 구분선을 추가한다.

## 요구사항
1. 구분선 추가
- 대상: `frontend/src/pages/SearchResults.jsx`
- 기존 구분선 스타일(`h-6 w-px bg-border`)과 동일하게 단계 필터 그룹과 유형 필터 그룹 사이에 구분선을 추가한다.

## 완료 기준
- 필터 UI에서 종료 버튼 오른쪽과 전체 유형 버튼 왼쪽 사이에 구분선이 표시된다.
- Docker 환경 `verify:fast`가 통과한다.
