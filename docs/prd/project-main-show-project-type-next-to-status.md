# 프로젝트 메인 상태 옆 프로젝트 종류 표시 PRD

## 배경
- 프로젝트 메인 상단에서 현재는 프로젝트 상태만 표시되고 있어, 종류 정보를 한눈에 확인하기 어렵다.

## 목표
- 프로젝트 상태 배지 오른쪽에 프로젝트 종류 배지를 추가한다.

## 요구사항
1. 종류 배지 추가
- 대상: `frontend/src/pages/BudgetProjectOverview.jsx`
- 프로젝트 제목 행에서 상태 배지 우측에 프로젝트 종류 배지를 표시한다.

2. 라벨 보정
- `project_type_label`이 있으면 우선 사용한다.
- 라벨이 없거나 영문 코드만 있는 경우 한국어 라벨로 보정해 표시한다.

## 완료 기준
- 프로젝트 상태 오른쪽에 프로젝트 종류가 표시된다.
- Docker 환경 `verify:fast`가 통과한다.
