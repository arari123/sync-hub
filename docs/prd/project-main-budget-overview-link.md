# 프로젝트 메인 예산 대비 집행 패널 전체 보기 링크 PRD

## 배경
- 프로젝트 메인 본문의 예산 대비 집행 패널 우측 상단에 예산 관리로 바로 이동하는 링크가 필요하다.

## 목표
- 예산 대비 집행 패널 상단 오른쪽에 `전체 보기` 링크를 표시한다.
- 링크 클릭 시 예산 관리 페이지로 이동한다.

## 요구사항
1. 링크 추가
- 대상: `frontend/src/pages/BudgetProjectOverview.jsx`
- 예산 대비 집행 카드 상단 오른쪽에 `전체 보기` 링크를 노출한다.

2. 이동 경로
- 링크 클릭 시 프로젝트별 예산 관리 경로(`/budget`)로 이동한다.

## 완료 기준
- `전체 보기` 링크가 예산 대비 집행 상단 오른쪽에 노출된다.
- 링크 클릭 시 예산 관리 페이지로 이동한다.
- Docker 환경 `verify:fast`가 통과한다.
