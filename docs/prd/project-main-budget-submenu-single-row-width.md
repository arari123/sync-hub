# 프로젝트 메인 예산 하위 메뉴 1행 고정 PRD

## 배경
- 프로젝트 메인 브레드크럼 패널의 예산 관리 하위 메뉴가 좁은 폭에서 줄바꿈되어 1행으로 보이지 않는다.

## 목표
- 하위 메뉴 드롭다운 폭을 확대하여 모든 항목(재료비 관리, 인건비 관리, 경비 관리)이 항상 1행으로 노출되게 한다.

## 요구사항
1. 드롭다운 폭 확대
- 대상: `frontend/src/pages/BudgetProjectOverview.jsx`
- 예산 관리 하위 메뉴 패널 폭을 확장해 3개 메뉴가 한 줄에 배치되도록 한다.

2. 줄바꿈 방지
- 하위 메뉴 텍스트가 줄바꿈되지 않도록 처리한다.

## 완료 기준
- 하위 메뉴 3개가 1행으로 표시된다.
- Docker 환경 `verify:fast`가 통과한다.
