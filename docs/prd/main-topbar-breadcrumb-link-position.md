# 메인 상단바 브레드크럼 위치/링크 개선 PRD

## 배경
- 메인 페이지 상단바에 브레드크럼이 추가되었으나, 현재 검색창 상단에 표시되고 경로 클릭 이동이 불가능하다.

## 목표
- 브레드크럼을 검색창 하단으로 이동한다.
- 브레드크럼 경로를 클릭하면 해당 페이지로 이동 가능하게 한다.
- 브레드크럼 가독성을 위해 폰트 크기를 기존 대비 소폭 확대한다.

## 요구사항
1. 브레드크럼 위치 변경
- 대상: `frontend/src/pages/SearchResults.jsx`
- 검색 입력창 하단에 브레드크럼을 배치한다.

2. 경로 클릭 이동
- 브레드크럼 각 경로를 클릭 가능한 링크로 제공한다.
- 경로 클릭 시 해당 라우트로 이동해야 한다.

3. 가독성 조정
- 브레드크럼 폰트 크기를 기존 대비 약간 키운다.

## 완료 기준
- 브레드크럼이 검색창 하단에 표시된다.
- 브레드크럼 경로 클릭 시 페이지 이동이 동작한다.
- Docker 환경에서 `verify:fast`가 통과한다.
