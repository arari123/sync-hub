# PRD: 메인 검색 상단 메뉴/무결과 상태/필터 UI 정리

## 목적
메인 검색 페이지의 상단 빠른 메뉴를 추가하고, 무결과 상태 노출 규칙을 정리하며, 필터 버튼 UI 일관성을 개선한다.

## 요구사항

### MSME-REQ-001: 상단 빠른 메뉴
- [AC-1] 상단바 유저 아이콘 왼쪽에 메뉴 아이콘을 추가한다.
- [AC-2] 메뉴 아이콘 클릭 시 앱 런처 형태의 메뉴 팝오버가 열린다.
- [AC-3] 메뉴 항목은 아이콘 + 라벨(아래 배치) 형태로 표현한다.
- [AC-4] 메뉴 구성은 `새 프로젝트 생성`, `데이터 허브(미구현)`으로 한다.

### MSME-REQ-002: 프로젝트 무결과 패널 규칙
- [AC-1] 프로젝트가 매칭되지 않았을 때 기존 `검색 조건과 일치하는 프로젝트가 없습니다.` 패널은 노출하지 않는다.
- [AC-2] 프로젝트 미매칭 + 문서만 검색된 상황에서는 위 패널이 보이지 않는다.

### MSME-REQ-003: 완전 무결과 패널
- [AC-1] 검색 후 프로젝트/문서 모두 결과가 없을 때만 `검색 결과가 없습니다.` 패널을 노출한다.

### MSME-REQ-004: 필터 UI 재정렬
- [AC-1] `내 프로젝트/전체 프로젝트`와 단계 선택 버튼을 같은 1행에 배치한다.
- [AC-2] 두 필터 그룹의 버튼 스타일을 통일감 있게 개선한다.
- [AC-3] 필터 패널 내 `프로젝트 생성` 버튼은 제거한다.
