# PRD: 프로젝트 메인 최신 안건 5개 + 전 페이지 배경 통일 (2026-02-15)

## 목적
- 프로젝트 메인에서 최신 안건을 빠르게 확인할 수 있도록 “마지막 안건” 영역을 최신순으로 최대 5개 노출한다.
- 전체 웹 페이지의 배경을 이슈관리(안건관리) 페이지와 동일한 배경(기본 body 배경)으로 통일한다.

## 범위
- 프로젝트 메인(프로젝트 개요) 화면의 “마지막 안건” 리스트.
- 메인(글로벌 검색) 화면 및 프로젝트 관련 주요 페이지의 페이지 루트 배경 처리.
- 안건 목록 API의 기본 정렬(프로젝트별 threads 목록).

## 요구사항
- REQ-001: 프로젝트 메인 “마지막 안건”은 최신순(최근 업데이트 기준)으로 최대 5개를 표시해야 한다.
- REQ-002: 모든 웹 페이지의 배경은 이슈관리(안건관리) 페이지의 배경과 동일해야 한다.
  - 배경 “색상” 기준은 페이지별 별도 배경(`bg-background`)이 아닌, `body` 기본 배경(그라데이션/그리드 포함) 노출을 기준으로 한다.

## 비범위
- 카드/패널 내부(콘텐츠 영역)의 배경(`bg-card`, `bg-white` 등) 스타일 변경.
- 안건 검색 로직/스코어링 로직 변경.

## 수용 기준
- AC-001: 프로젝트 메인에서 안건이 5개 초과 존재할 때 “마지막 안건”이 최근 업데이트 순으로 5개만 노출된다.
- AC-002: `/home`, `/project-management/projects/:id`, `/project-management/projects/:id/budget` 페이지의 배경이 이슈관리 페이지(안건관리)와 동일하게 보인다.

- `frontend/src/pages/BudgetProjectOverview.jsx`
