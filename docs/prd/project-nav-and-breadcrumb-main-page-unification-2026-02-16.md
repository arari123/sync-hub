# PRD: 프로젝트 메뉴/브레드크럼 명칭 통일 및 브레드크럼 바 높이 정렬 (2026-02-16)

## 목적
- 프로젝트 관련 화면의 메뉴/브레드크럼 용어를 일관되게 통일한다.
- 메인 페이지와 프로젝트 페이지의 브레드크럼 상단바 높이 인상을 맞춰 UI 일관성을 높인다.

## 적용 범위
- 프로젝트 컨텍스트 메뉴: `frontend/src/components/ProjectContextNav.jsx`
- 프로젝트 메인/예산 메인 커스텀 헤더: `frontend/src/pages/BudgetProjectOverview.jsx`, `frontend/src/pages/BudgetProjectBudget.jsx`
- 프로젝트 하위 페이지 브레드크럼: `Agenda*`, `BudgetProject*`, `ProjectPlaceholderPage`
- 홈 브레드크럼 상단바: `frontend/src/pages/SearchResults.jsx`

## 요구사항
- REQ-001: 메뉴 명칭 `이슈 관리`를 `안건 관리`로 변경한다.
- REQ-002: 프로젝트 메인/예산 메인/안건 관리/일정 관리/사양 관리/데이터 관리/프로젝트 설정 페이지 브레드크럼 첫 항목의 `프로젝트 관리`를 `메인 페이지`로 변경한다.
- REQ-003: 메인 페이지 브레드크럼 상단바와 다른 페이지 브레드크럼 상단바의 세로 폭 인상을 통일한다.

## 수용 기준
- AC-001: 상단 프로젝트 메뉴에서 `안건 관리`가 노출되고 `이슈 관리`는 노출되지 않는다.
- AC-002: 해당 프로젝트 페이지들의 브레드크럼 첫 항목이 `메인 페이지`로 표시된다.
- AC-003: 홈 브레드크럼 바가 프로젝트 페이지 브레드크럼 바와 유사한 높이/패딩 톤으로 보인다.
